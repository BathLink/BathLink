# BathLink

Bathlink is a mobile app designed to help connect people through mutual activities. This respository contains the react native frontend and the aws cdk backend.

## Usage

### Frontend

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

### Backend

```bash
    cd Backend/cdk
    cdk deploy
   ```
